Blockchain

What is Blockchain?

Definition

Blockchain is a technology used to process any transaction of value using a decentralized (P2P to be specific) network of computers spread across the world.

Characterstics:

  1. Works by a P2P network model.
  2. A transaction is called to any procedure involing debit, credit and exchange of any entity of value, hence not limited to only financial transactions.
  3. Every transaction is secured from hacking attempts.
  4. Works on a completely decentralized design; i.e., there is NO provision to a central authority keeping track of all things.
  5. Every computer on the P2P network shared the database of every transactions.

How it works?

  1. Someone initiates a transaction.
  2. The transaction and user's status are validated for verification.
  3. After validation a transaction might invlove some crypto-currency like Bitcoin being used.
  4. On successful validation a empty block is generated for the transaction.
  5. This generated block is added to the exiting blockchain of other transactions (which are not limited to only financial transactions, as a transaction could be supposed for any entity of value. Please note that in this context it's not important that how blocks are added to each blockchain and how each block stores the transactions of the users; what important is that each block stores some transactions which are added to the blockchain.)
  6. Once written to the block the transaction is complete and immutable to any kind of changes.
